Business Project / 중국용 Search.Mol-In 웹 서비스

개요

중국에서 서비스 하기위한 Searh.Mol-In 웹 서비스입니다. 기존 서비스와 동일하게 화학 데이터를 제공하고있습니다. 회원 등록하고 라이센스를 발급받으면 범위 내에서 화학 데이터를 열람하고 다운로드 받을 수 있습니다. 중국 외 IP는 기존 Search.Mol-In 웹 서비스로 Redirect 시킵니다.

  • China Search.Mol-In(중국 IP로만 접근 가능)

  • 유지 보수
  • 역할: Back-End, Front-End 기능 개발
  • 사용 기술: Spring, MariaDB, PostgreSQL
  • 개발 언어: Java, JavaScript, HTML

상세

환경

  • Spring v3.1.1
  • Java v1.6
  • MySQL v5.7.44

담당 개발 기능

  • 중국 외 IP 인터셉터 처리 (기존 Search.Mol-In으로 Redirect)
  • 디자인 수정 반영, 이슈 수정 등 유지보수
