Business Project / 사내 관리자 사이트

개요

Mol-Instincts, Search.Mol-In 서비스의 관리자 사이트와 Roppor 서비스의 관리자 사이트를 기획, 디자인, 개발까지 혼자 맡아 완성했습니다. 단,기획에 관해서는 실제 사용할 고객 담당 직원과 논의하며 요청 사항을 받아 변경했습니다. 사내 서버에 독립적으로 구축했으며, 주로 구매 기록과 다운로드 기록을 관리하는 용도로 쓰였습니다.

  • 개발 기간: 22.10-23.02
  • 역할: 기획, 디자인, Back-End, Front-End 기능 개발
  • 사용 기술: Spring Boot, MariaDB
  • 사용 언어: Java, JavaScript, HTML

상세

Mol-Instincts, Search.Mol-In Admin 환경

  • Spring Boot v2.7.2
  • Java v1.8
  • MariaDB v10.6.14

Mol-Instincts, Search.Mol-In Admin 개발 기능

  • 데이터 구매 기록 관리
  • 구매 기록 차트 (chart.js)
  • 문의 메세지 관리
  • 데이터 구매 메일 재발송 기능
  • 구매 오류 기록 관리 및 담당자 메일 발송

Roppor Admin 환경

  • Spring Boot v1.4.7
  • Java v1.8
  • MariaDB v10.6.14

Roppor Admin 개발 기능

  • 다운로드 메일 발송 기록, 실제 다운로드 기록 수집
  • 공지사항 작성 기능 (CRUD)
