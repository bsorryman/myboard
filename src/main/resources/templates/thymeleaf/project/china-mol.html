Business Project / 중국용 Mol-Instincts 웹 서비스

개요

중국에 서비스하기 위한 Mol-Instincts 웹 서비스입니다. 중국 사정에 맞게 기존 Mol-Instincts에서 기능을 수정, 삭제하며 개발했습니다. 중국에서는 S3를 쓸 수 없어, 첨부 파일은 로컬 서버로 업로드하게 하고 이중화로 관리했습니다. 검색 및 화합물 상세 페이지는 없으며 Data Request(화학 데이터 계산 요청 서비스) 기능은 유지하고 있습니다.

  • China Mol-Instincts (중국 IP로만 접근 가능)

  • 개발 기간: 22.04-22.05
  • 역할: Back-End, Front-End 기능 개발
  • 사용 기술: Spring Boot, MariaDB, PostgreSQL, AWS SES
  • 사용 언어: Java, JavaScript, HTML

상세

환경

  • Spring Boot v4.3.9
  • Java v1.8
  • MariaDB v10.6.14

담당 개발 기능

  • 기존 Mol-Instincts 서비스 복제
  • AWS SES를 활용한 Data Request 메일 발송 기능
  • 첨부파일 로컬 서버 업로드 기능(S3 대체)
  • 첨부파일 서버 이중화 적용 (Rsync Script)
  • 중국 외 IP 인터셉터 처리 (기존 Mol-Instincts로 Redirect)
  • 랜딩 페이지 서버 반영
