Business Project / Roppor 웹 서비스

개요

드론 군집 소프트웨어인 Roppor 관련 웹 서비스입니다. 주로 Roppor 관련 랜딩 페이지로 이뤄져있으며, 소프트웨어 다운로드 링크를 메일로 발송하는 기능이 있습니다. Static 페이지 위주로만 관리하는 서버와 공지사항, 블로그 포스팅, 다운로드를 관리하는 API 서버로 나뉘어있습니다.

  • Roppor

  • 유지보수
  • 역할: Back-End, Front-End 기능 개발
  • 사용 기술: Spring Boot, MariaDB, AWS EC2/SES
  • 사용 언어: Java, JavaScript, HTML

상세

환경

  • Spring Boot v1.5.10
  • Java v1.8
  • MariaDB v10.6.14

담당 개발 기능

  • 소프트웨어 다운로드 링크 이메일 발송 기능
  • 이메일 검증 API 적용 (MailBoxValidator)
  • 랜딩 페이지 서버 업로드

다운로드 링크 발송 및 이메일 검증
