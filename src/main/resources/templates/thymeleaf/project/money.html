University Project / 통합 금융상품 검색 웹 서비스

개요

직접 만든 금융 상품 DB 내에서 상품을 검색하고 비교하는 대학교 실습 프로젝트입니다. 예금, 적금 등 상품을 분류하고, 회원가입 시 즐겨찾기 기능을 통해 목록을 저장하도록 했습니다. 또한 회원가입 시 입력했던 정보를 기반으로 나이, 성별, 직업 별로 어떤 상품들을 가장 많이 즐겨찾기 했는지 통계를 내어 원 그래프로 비교할 수 있습니다.

  • 개발 기간: 19.09-19.12
  • 프로젝트 인원: 4인
  • 역할: 조장, 기능 개발, 호스팅
  • 사용 기술: Wordpress, MySQL, cafe24
  • 사용 언어: PHP, JavaScript, HTML

상세

환경

  • Wordpress
  • MySQL

담당 개발 기능

  • Wordpress Admin을 통한
  • PHP로 DB 접속 환경 및 쿼리문 설계
  • Javascript 및 HTML로 검색 화면 기능 개발
  • 회원가입/로그인/회원 정보 관리
  • 즐겨찾기 및 통계 기능
