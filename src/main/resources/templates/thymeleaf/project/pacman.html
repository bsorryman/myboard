University Project / [프로젝트] 팩맨 강화학습 시뮬레이션

개요

유전 알고리즘/인공신경망을 활용한 팩맨 강화학습 시뮬레이션

  • 유전 알고리즘(Genetic algorithm)

    • 팩맨 집단은 한 세대에 16마리의 인구수(population)을 가진다.
    • 한 세대의 게임이 끝나면 적합도(fitness)를 기준으로 교배(crossover)/변이(mutation) 연산을 거친다.

  • 인공신경망(neural network)

    • 팩맨 각 개체의 데이터
    • Input 값으로 자신의 위치에서 이동 가능한 방향의 데이터를 받는다.
      (고스트 인식)
    • 인공신경망의 연산을 거친 후, Output 값으로 이동할 방향 선택한다.
