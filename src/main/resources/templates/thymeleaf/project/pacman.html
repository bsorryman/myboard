University Project / 팩멘 게임 강화학습 시뮬레이션

개요

팩맨이 스스로 게임을 하고 시간이 지날수록 게임 방법을 학습하는 시뮬레이션 데스크톱 실행 프로그램입니다. 유전알고리즘(Genetic Algorithm)과 인공신경망(Neural Network)를 활용해 강화 학습을 구현했습니다. 사용자가 시뮬레이션을 종료하면 게임 결과 데이터가 csv파일로 export됩니다. 최종 결과물은 60세대 이상 진화했을 때, 테스트 결과 사람보다 게임을 대체로 잘한다는 결론이 나왔습니다. 관련하여 학술지 논문을 제 1저자로 게재했습니다.

  • 학술지 논문

  • 개발 기간: 20.03-20.6
  • 프로젝트 인원: 3인
  • 역할: 조장 및 기능 개발
  • 개발 언어: Python

상세

유전 알고리즘

팩맨은 한 세대에 16마리의 인구 수를 가지며, 한 세대의 모든 게임이 끝나면 16마리 중 게임 결과가 잘나왔던 팩맨들을 토대로 교배 연산 및 변이 연산을 거쳐 새로운 다음 세대의 16마리 팩맨을 탄생시킵니다. 연산의 대상이 되는 데이터는 각 팩맨의 핵심인 인공신경망입니다.

인공신경망

게임을 진행하는 팩맨 자체가 인공신경망입니다. Input 값으로 자신이 위치한 곳에서 움직일 수 있는 방향의 데이터를 받습니다. 각 방향에 팩맨이 위치하고 있는지 인식하고, Output 값으로 움직여할 방향을 선택합니다. 움직일 때마다 매순간 인식하고, 판단하며, 움직입니다.

담당 개발 기능

  • 유전 알고리즘 설계
  • 인공신경망 설계
  • 팩맨의 인식 기능 개발
  • 게임 결과 csv 파일 export 기능 개발
