Business Project / IKC File Supply System

개요

중국 업체 요청으로 개발하게 된 데스크톱 앱입니다. 네트워크 필요 없이 프로그램 실행 만으로 사용할 수 있도록 Stand Alone하게 만들어졌습니다. 중국 쪽에서 요청한 특정 범위의 200만개 화학 데이터를 포함한 DB를 내장하고 있습니다. 화합물 검색 및 기본 물성 정보를 참고할 수 있으며, IKC File 이라는 핵심 화학 데이터를 다운로드할 수 있습니다.

  • 개발 기간:23.05-23.07
  • 역할: Back-End 및 Front-End 기능 개발
  • 사용 기술: Electron, SQLite
  • 사용 언어: JavaScript, HTML

상세

환경

  • Electron v23.2.0
  • Electron Forge v7.2.0
  • Node.js v18.16.0
  • SQLite v3.34.0

담당 개발 기능

  • SQLite DB 설계 및 쿼리문 작성
    • FTS 테이블 및 쿼리문 구성 (Full Text Search)
    • IKC 파일 및 이미지 데이터 바이너리 컬럼 구성
  • Web Worker 기능 (멀티스레드)
    • 단일 스레드인 Electron을 멀티스레드로 활용
    • 검색 시간 소요 시 교착 상태가 발생하던 이슈 해결(응답없음 해결)
  • 검색 화면 구성 기능
    • Refresh없는 비동기 검색 결과 화면 구현
  • Window Installer 패키징 (.msi)
