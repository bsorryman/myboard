Business Project / Search.Mol-In 웹 서비스

개요

화학 DB 중에서도 이미 데이터 계산이 완료된 심층 데이터를 관리하는 Mol-In DB 관련 서비스를 제공하는 웹입니다. 500만개의 심층 화학 데이터를 바탕으로 검색이 가능하며, 회원등록 후 라이센스를 발급받으면 데이터를 열람할 수 있습니다. 하나의 화합물 당 제공하는 물성 데이터의 종류는 약 89개입니다.

  • 사용 언어: Java, JavaScript, HTML
