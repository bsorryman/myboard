개요

화학 정보 웹 서비스

  • 자사 화학 데이터 관련 랜딩 페이지
  • 9천만 화학 DB 데이터 제공
  • 유료 화학 데이터 결제 기능
  • 고객 요청 화학 데이터 계산/생성 서비스

도메인

  • www.molinstincts.com: 기본 화학 DB 데이터 제공
  • payment.molinstincts.com : 결제 서버

서브 도메인

  • search.molinstincts.com: 심층 화학 DB 데이터 제공
