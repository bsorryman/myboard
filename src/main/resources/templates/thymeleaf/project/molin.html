Business Project / Mol-Instincts 웹 서비스

개요

화학 데이터 및 관련 랜딩 페이지를 제공하는 웹 서비스입니다. 보유한 9천만개 화학 데이터를 기반으로 검색 및 데이터를 제공합니다. 그중 심층 데이터를 보유한 500만개 대상으로는 구매를 통해 제공하고 있습니다. 제공하는 모든 화학 데이터 상세 페이지는 SEO로 관리되고 있습니다. DB에 없는 데이터는 Data Request 기능을 통해 직접 계산을 요청할 수 있습니다.

  • Mol-Instincts

  • 개발 기간: 21.07-21.10
  • 역할: Back-End, Front-End 기능 개발
  • 사용 기술: Spring Boot, MariaDB, PostgreSQL, AWS EC2/S3/SES, Elasticsearch, Paygate, Paypal
  • 사용 언어: Java, JavaScript, HTML

상세

환경

  • Spring Boot v4.3.9
  • Java v1.8
  • MariaDB v10.6.14
  • PostgreSQL v9.5.8

담당 개발 기능

  • Elasticsearch 검색 기능(쿼리문/ 결과화면 적용)
  • Search.Mol-In 서비스와 API 결제 연동
  • 화합물 페이지 내에 Search.Mol-In 서비스와 Iframe 연동
  • AWS SES/S3를 활용한 Data Request 기능 개발
  • 캡챠 기능
  • Lucy 필터 적용 (XSS 방지)
  • 이메일 검증 API 적용 (MailBoxValidator)
  • 랜딩 페이지 서버 반영
