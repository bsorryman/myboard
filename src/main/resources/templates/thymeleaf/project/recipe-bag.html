Toy Project / Recipe Bag

개요

레시피 데이터 제공 데스크톱 앱

  • 13,000개 레시피 데이터 및 이미지 검색/열람/다운로드

설치파일 및 Git

  • RecipeBag-1.0.0 Setup.exe
  • GitHub

개발 환경

  • Electron v28.1.3
  • ELectron forge v7.2.0
  • Node.js v18.18.0
  • SQLite3 v3.39.0
  • Jquery v3.7.1

개발

담당 역할

  • 2인 개발
  • 데이터 수집 및 DB 설계
  • Back-End/Front-End 기능 개발

개발 내역

개발 기간 개발 내용 사용 기술
2024.01 - 2024.04
  • DB 구성
    • kaggle - CSV 파일을 활용한 DB 테이블 생성
    • 파일 다운로드, 이미지 표시를 위한 파일 압축 및 바이너리 컬럼 삽입 자동화
    • 빠른 검색 및 성능을 위한 FTS 테이블 설계(Full Text Search)
  • IPC 통신 방식 설계
    • Back단의 Main Process(Node.js)와 Front단의 Renderer Process(Chromium)간 데이터 통신
  • 검색 기능 개발 및 검색 결과 최적화
  • Web Worker를 활용한 검색 성능 개선(멀티스레딩)
    • 단일 스레드인 Electron을 멀티스레드로 활용
    • 검색 시간 소요 시 교착 상태가 발생하던 이슈 해결(응답없음 해결)
  • All Recipe 기능(모든 레시피 데이터)
  • 압축 파일 다운로드 기능 개발(레시피 이미지, 텍스트)
  • 검색어 Highlight 기능(레시피 상세 모달창)
  • 최근 검색어 저장 기능(Appdata 내 JSON 파일 관리)
  • 타이틀 메뉴 기능 개발
