Toy Project / Recipe Bag

개요

약 13,000개 대상의 레시피를 검색 및 열람하고 다운로드 받을 수 있는 데스크톱 실행 프로그램입니다. 데이터는 Kaggle의 무료 데이터 및 이미지를 활용하여 모두 영문으로 구성되어있습니다.

  • Git: recipe.electron
  • 설치파일: RecipeBag-1.0.0 Setup.exe

  • 개발 기간: 24.01-24.04
  • 프로젝트 인원: 2인
  • 역할: 기획, 기능 개발
  • 사용 기술: Electron, Node.js, SQLite3
  • 사용 언어: JavaScript, HTML

상세

환경

  • Electron v28.1.3
  • ELectron forge v7.2.0
  • Node.js v18.18.0
  • SQLite3 v3.39.0
  • Jquery v3.7.1

DB 구성

  • SQLite3
  • kaggle - CSV 파일을 변환하여 DB에 삽입
  • 파일 다운로드, 이미지 표시를 위한 파일 압축 및 바이너리 컬럼 삽입 자동화
  • 빠른 검색 및 성능을 위한 FTS 테이블 활용(Full Text Search)

기능 개발

  • IPC 통신: Back단의 Node.js로 이루어진 Main Process와 Front단의 Chromium의 Renderer Process간 데이터를 통신
  • Web Worker의 Worker Thread 통해 단일 스레드인 Main Process를 멀티 스레드로 활용('응답 없음' 방지)
  • 검색 결과 무한 스크롤
  • All Recipe 클릭 시 모든 레시피 열람 가능
  • 레시피 이미지와 텍스트 데이터가 압축된 파일 다운로드
  • 레시피 View 내에 검색어 Highlight
  • JSON 파일을 저장토록 하여 최근 검색어 저장
  • 일렉트론 앱의 기본 타이틀 바 메뉴를 제거하고 새로이 추가 - 최소화, 최대화, 닫기
